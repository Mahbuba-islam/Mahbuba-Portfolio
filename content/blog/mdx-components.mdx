Plain Markdown is great for content, but every now and then you want a real
React component inside the page — a styled callout, a captioned image, a
keyboard hint. MDX gives you both, in the same file.

## Keyboard hints

Press <Kbd>⌘</Kbd> + <Kbd>K</Kbd> on the blog index to focus the search box.

## Plain Markdown still works

Lists, **bold**, _italic_, [links](https://nextjs.org), and code blocks all
behave exactly like in `.md`:

```ts
export function add(a: number, b: number) {
  return a + b;
}
```

That's the whole point — MDX is a superset, not a replacement.
